js中关于数组循环的各种方法的区别

发表于 2017-02-15   |   分类于 javascript   |  

js中对数组循环的方法有很多，都有各自的用途，但是经常会混淆用错，下面列出各种方法的区别与常用场景。

方法名 方法描述 返回值 函数接受参数 用途(参考)
map 对数组的所有成员依次调用一个函数，根据函数结果返回一个新数组。 返回新数组,原数组不变 当前成员、当前位置、数组本身 适合数组值变动，返回新数组
forEach 遍历数组的所有成员，执行某种操作 没有返回值 当前成员、当前位置、数组本身 适合不需要返回值的数组遍历，如改动数组里的值
filter 所有数组成员依次执行该函数，返回结果为true的成员组成一个新数组返回 返回新数组,原数组不变 当前成员(、当前位置、数组本身) 筛选数组，返回新数组
some/every 判断数组成员是否符合某种条件。some有真则返回true，every所有真才返回true 返回布尔值 当前成员、当前位置、数组本身 判断数组成员是否满足某种条件，返回布尔值
reduce/reduceRight 依次处理数组的每个成员，最终累计为一个值。 返回一个值(string/num) 累积变量、当前变量(、当前位置、数组本身) 判断数组成员是否满足某种条件，返回布尔值
  • 这几个方法里面，function是匿名函数，this指向的是window。如果需要使用当前环境，需要传入第二个参数来控制。
  • forEach不可中断，如果需要中断，请使用for
  • reduce方法第二个参数可以传入初始值。建议这样使用，防止空数组取不到值出错！
  • 方法也可以扩展，例：找出数组中最长的成员！也可以用reduce方法。
