forEach 方法为数组中含有有效值的每一项执行一次 callback 函数，已删除或者未赋值的项将被跳过。callback 函数会被依次传入三个参数：

  • 数组当前项的值
  • 数组当前项的索引
  • 数组对象本身

新方法 for-of

为解决for-in与forEach的问题。在ES6中引入了新方法 for-of 专注解决数组遍历问题。 以后for-in可以专心遍历对象的属性了~

总结

以上四种for循环方法各有优劣，应在实际项目中多加考虑。
